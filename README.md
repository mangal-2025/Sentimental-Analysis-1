## üåü Project Overview

This repository contains a simple, yet effective, Python script that performs **Sentiment Analysis** on text data read directly from an Excel spreadsheet. 

It utilizes the popular **VADER (Valence Aware Dictionary and sEntiment Reasoner)** model from the **NLTK (Natural Language Toolkit)** library to determine the emotional tone (Positive, Negative, or Neutral) of each entry.

The workflow is straightforward:
1. **Input:** Reads data from `CRM_v2.xlsx`.
2. **Analysis:** Runs the sentiment model on the designated text column.
3. **Output:** Saves the original data along with new sentiment scores and labels to `feedback_analyzed.xlsx`.

### **Execution Steps**
**Place the files**: Ensure both excel_analyzer.py and your input file (e.g., feedback.xlsx) are in the same folder.

**Verify the Column**: Crucially, make sure the TEXT_COLUMN variable inside the excel_analyzer.py script matches the name of the column in your Excel file that contains the reviews (e.g., 'Review_Text').

**Run the script**: Open your Terminal, navigate to the project folder using the cd command, and execute the script:

Bash

python3 excel_analyzer.py

**Check Results**: A new file named feedback_analyzed.xlsx will be created in the same folder, containing the results.
